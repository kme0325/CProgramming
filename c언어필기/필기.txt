솔루션 : 프로젝트들을 하나로 묶은 것

프로젝트 : 실행 파일(exe) 하나를 만들어내기 위해
필요한 여러 개의 소스 파일과 헤더 파일을 하나로 묶어 놓은 것

소스 파일 : 프로그램 코드를 작성한 파일(.c)
	  프로그램을 저장하는 파일				

헤더 파일 : 소스 파일에 포함시킬 파일(.h)
	  #include 명령어를 통해 추가합니다.
	  프로그램에서 사용할 다양한 선언을 보관

디렉터리 : 컴퓨터의 폴더(Folder)와 사실상 같은 개념
	 파일 시스템적으로 파일을 묶어둡니다.
	 주로 도스, 리눅스 등에서는 디렉토리
	 윈도우, 매킨토시 등에서는 폴더라고 부르기도 합니다.	

디버그(Debug) : 컴퓨터 프로그램 개발 단계 중에 발생하는 시스템의 논리적인
	        오류나 비정상적인 연산을 찾아내고 수정하는 작업 과정
